Network Infrastructure Implementation for New Leaf Hospital

As part of the project team, I was responsible for configuring network devices, implementing security measures, and performing testing and verification to ensure the network infrastructure's stability and performance.

Technical Skills Utilized:

  • Network Architecture Design
  • Cisco Switching Configuration
  • VLAN Implementation
  • Active Directory Integration
  • Network Security Implementation
  • Wireless Connectivity Configuration
  • VoIP Phone System Implementation
  • Server Systems Configuration
  • Network Management Tools Configuration

Project Outcomes:

  • Core Network Services: Active Directory domain controllers for centralized user/device authentication, DHCP and DNS services for IP management.
  • Network Security: Perimeter firewalls at each campus location for traffic filtering and security, secure access controls, firewalling, and threat prevention measures.
  • LAN/WAN Infrastructure: Layer-2 LAN switching for local area connectivity, inter-site routing and WAN links for communication between campuses, VLANs to logically segment different service traffic.
  • Wireless Connectivity: Wireless access points for comprehensive WiFi coverage across all campuses.
  • Unified Communications: VoIP phone system with centralized PBX/call control, enabling inter-campus voice/video communication.
  • Server Systems: Web servers for hosting hospital websites and applications.
  • Network Management: Network monitoring tools to track performance and availability, antivirus/anti-malware protection.

Project Preparation:

  • Site Visits: We conducted site visits to assess the physical environment and infrastructure at each hospital location.
  • Equipment Procurement: We procured equipment based on the network architecture design and equipment specifications.
  • Configuration and Installation: We configured network devices and installed network equipment in designated locations.
  • Security Implementation: We implemented security measures like firewalls, intrusion detection systems, access controls, encryption protocols, VPNs, and secure authentication mechanisms.
  • Testing and Verification: We developed test plans and procedures to verify the performance, stability, and security of the network infrastructure.
  • Training and Knowledge Transfer: We provided training sessions and workshops for hospital IT staff.
  • Ongoing Support and Maintenance: We established service level agreements (SLAs) and performed regular maintenance activities.
